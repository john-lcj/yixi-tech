使用说明

给非工程师同事看的速查指南

🚀 三步流程
  1. 在「全局信息 / 首页 Hero / 公司地址」中编辑字段，点击 💾 保存草稿（仅保存到本地浏览器，不会动线上）。
  2. 到「导出更新版本」点击 📦 应用并下载 ZIP，得到一个补丁包。
  3. 解压后用 OSS 控制台或 ossutil 原样上传覆盖，刷新 CDN 即可生效。
⚠️ 注意事项
  • 本后台依赖浏览器同源 fetch，请通过 HTTP 访问（如 http://127.0.0.1:8000/admin/ 或正式域名 https://yourdomain/admin/），不要用 file:// 直接打开。
  • 「全局信息」用字符串替换，请不要随意改「当前值」字段，它必须与线上实际文本一致。建议先点 🔄 从线上读取当前值。
  • 页脚由 snippets/footer-zh.html / snippets/footer-en.html 配合 scripts/sync-footers.py 维护——本后台修改电话/邮箱/ICP 后，请同步更新这两个片段，避免下次重新同步覆盖你的修改。
  • 更复杂的内容（产品、新闻、应用案例）建议直接编辑源 HTML，再走 git 部署。
🔐 修改后台密码

编辑 admin/index.html 中常量 const PASS = '…'; 即可。生产环境建议把 admin/ 目录从公开访问中剔除（不上传 OSS，或在 CDN 上加访问控制）。
